9. **A few random tips (if time)**
- Adding to the most recent commit or changing the commit message
```shell
git commit --amend
```
- Undoing your last commit 
```shell
git reset HEAD~1
```
- Overwrite the remote with whatever is on the local (dangerous)
```shell
git push --force
```
